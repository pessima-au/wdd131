Site Plan for NASA Photo of the Day App

Site Name

Name: NASA Photo of the Day

Reason: The name highlights the app’s purpose of sharing NASA’s daily featured images, making it concise and engaging for space enthusiasts.

Site Purpose

The purpose of this site is to provide a visually stunning and educational platform where users can explore NASA’s Astronomy Picture of the Day (APOD), learn about the universe, and access daily updates with high-quality images and descriptions.

Scenarios

  • What was NASA’s featured image on my birthday?
  • How can I learn more about the science behind each daily image?

Color Schema

#000
#3a86ff
#1a1a1a
aqua

Usage:

  • #000: Background color for the header and footer.
  • #3a86ff: Accent color for headers, links, and buttons.
  • #1a1a1a: Background for content sections like the hero and photo containers.
  • aqua: Highlight color for interactive elements like buttons and descriptions.

Typography

  • Montserrat: Used for headers and titles to create emphasis and clarity.
  • Cabin: Used for body text to ensure readability and simplicity.
